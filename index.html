🎯 Objetivo

Separar el comportamiento común de las características específicas de cada GAD.

Ingreso_GAD = Comportamiento_Común + Efecto_Específico

🧠 Modelo Completo (CON efectos GAD)

Captura el comportamiento específico de cada GAD incluyendo sus características únicas.

ln(Ingreso) = β₀ + β₁·lag1 + β₂·lag2 + β₃·lag3 + α_GAD + ε

📏 Modelo Base (SIN efectos GAD)

Establece una línea base neutral que representa el comportamiento "promedio" común a todos los GADs.

ln(Ingreso) = β₀ + β₁·lag1 + β₂·lag2 + β₃·lag3 + ε

⚖️ Cálculo del Factor de Corrección

El factor de corrección ajusta las predicciones del modelo base según las características específicas de cada GAD.

Predicción_Corregida = Predicción_Modelo_Base × Factor_Corrección

🎯 ¿Por qué funciona este enfoque?

Estabilidad: El modelo base proporciona una tendencia estable y confiable

Personalización: El factor ajusta esa tendencia a las características específicas del GAD

Robustez: Combina la generalización del modelo base con la precisión local del factor

📊 Interpretación del Factor 1.49

Este GAD tiene características que lo hacen sistemáticamente más eficiente en la generación de ingresos:

  • Mayor capacidad de recaudación
  • Mejor gestión administrativa
  • Ubicación estratégica o características económicas favorables
  • Mayor cultura de pago ciudadana

Resultado: Sus ingresos reales superan consistentemente las predicciones del modelo promedio en aproximadamente 49%.

💡 ¿Por qué dos modelos?

Modelo Completo: Captura la realidad completa pero puede "sobreajustar" a características específicas que pueden no persistir en el futuro.

Modelo Base: Proporciona una referencia estable y neutral que sirve como "denominador común" para todos los GADs.

Factor de Corrección: Permite personalizar las predicciones manteniendo la estabilidad del modelo base.
